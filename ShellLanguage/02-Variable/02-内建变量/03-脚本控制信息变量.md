# 03.Shell 脚本控制信息变量

特殊变量保存了脚本运行时的一些控制信息，是只读变量。
---------------------------------------------------------------------------------------------------
$?     
上一个代码或者shell程序在shell中退出的情况，如果正常退出则返回0，反之为非0值。
使用  echo $?  可以查看命令执行退出状态，成功还是失败。 

$$     
本程序的(进程ID号)PID
$ echo $$
3340
$ ps -p $$
  PID TTY          TIME CMD
 3340 pts/1    00:00:00 bash

$!     
上一个命令的PID
